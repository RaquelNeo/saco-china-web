## VISUAL STYLE
- Background: white or very light cream
- Typography: large and bold for headings, clean and readable for body
- Colors: black, white, warm neutral or slate as accent — NO bright colors
- Lots of whitespace — nothing crowded
- Minimal top navigation
- Feel: premium European brand with Asian manufacturing excellence

---

## TECH STACK
- HTML + CSS + vanilla JavaScript (no frameworks)
- Mobile responsive
- Fast loading
- SEO optimized with schema.org structured data for LLM search engines (ChatGPT, Claude, Perplexity)
